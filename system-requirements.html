Test Your Device
Minimum Requirements

Basic Playable Experience

These specifications will allow you to play Bloxd with basic graphics and performance.

  • Operating System: Windows 7+ (64-bit), macOS 10.12+, Ubuntu 18.04+
  • Browser: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+ (latest versions recommended)
  • RAM: 2 GB minimum, 4 GB recommended for smooth gameplay
  • Internet: Stable broadband connection (1 Mbps minimum, 5+ Mbps recommended)
  • Graphics: Any modern integrated graphics (Intel HD 4000+, AMD Radeon HD 7000+)
  • Storage: No installation required (browser-based), 100 MB cache space
  • Display: 1024x768 minimum resolution, 1920x1080 recommended
  • Audio: Any sound card with speakers or headphones
Recommended Requirements

Optimal Gaming Experience

For the best performance and visual quality, we recommend these specifications.

  • Operating System: Windows 10+, macOS 11+, or Linux (latest)
  • Browser: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
  • RAM: 4 GB or more
  • Internet: Fast broadband connection (5+ Mbps recommended)
  • Graphics: Dedicated graphics card or modern integrated graphics
  • Processor: Intel i3 or AMD equivalent (2.0 GHz+)
Mobile Requirements

Mobile & Tablet Gaming

Bloxd works great on mobile devices with these specifications.

  • iOS: iPhone 6s+ or iPad 5th gen+ (iOS 12+)
  • Android: Android 6.0+ with 2GB RAM
  • Browser: Safari (iOS) or Chrome (Android)
  • Internet: 3G or better connection
  • Touch: Multi-touch support recommended
